# Badran CNC — نظام إدارة المبيعات 🏭

تطبيق سطح مكتب احترافي لإدارة المبيعات والمخزون مبني بـ Electron.

---

## 📁 ملفات المشروع

```
badran-cnc/
├── index.html       ← واجهة التطبيق الكاملة
├── main.js          ← Electron Main Process
├── preload.js       ← جسر آمن بين الواجهة والـ Node.js
├── package.json     ← إعدادات المشروع
├── logo-badran.png  ← اللوجو (أضفه بنفسك)
└── logo-badran.ico  ← أيقونة Windows (أضفها بنفسك)
```

---

## 🚀 كيف تشغّل التطبيق (للتطوير)

### المتطلبات
- [Node.js](https://nodejs.org) نسخة 18 أو أحدث
- npm (بييجي مع Node.js تلقائياً)

### الخطوات

```bash
# 1. افتح مجلد المشروع في Terminal/CMD
cd badran-cnc

# 2. تثبيت المكتبات
npm install

# 3. تشغيل التطبيق
npm start
```

---

## 📦 كيف تعمل ملف .exe جاهز للتوزيع (Windows)

```bash
# تأكد أن logo-badran.ico موجود
npm run build-win
```

الملف النهائي هيظهر في مجلد `dist/` وهيكون installer كامل.

## 💾 أين تتحفظ البيانات؟

البيانات بتتحفظ **على جهاز المستخدم مباشرة** في:

- **Windows:** `C:\Users\[اسم المستخدم]\AppData\Roaming\badran-cnc\badran_data.json`
- **Mac:** `~/Library/Application Support/badran-cnc/badran_data.json`
- **Linux:** `~/.config/badran-cnc/badran_data.json`

> ✅ البيانات **مش بتتمسح** حتى لو حدّثت التطبيق.  
> ✅ عشان تعمل **نسخة احتياطية** افتح الملف ده وانسخه في مكان تاني.
